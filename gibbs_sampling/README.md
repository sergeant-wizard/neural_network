# ギブス・サンプリング

## 目的
2ユニットの簡単なボルツマシンでギブス・サンプリングを実施し、
期待通りの振る舞いをするかを検証する。

## 理論
テキスト(8.8)式に用いてギブス・サンプリングを実施。

## 結果

下記通り、3通りの重みとバイアスに対してギブス・サンプリングを実施することで期待通りの確率分布を得ることができた。

### W = 0, b = 0 のとき

期待通り、等方的な分布が得られた。

x0 | x1 | p(x) (numerical) | p(x) (theoretical)
---|----|------------------|-------------------
0  | 0  | 24.9695          | 25
0  | 1  | 25.0000          | 25
1  | 0  | 25.1617          | 25
1  | 1  | 24.8688          | 25

### W01 = 1, b = 0 のとき

`p(x_0 = 1, x_1 = 1) = exp(1) / (3exp(0) + exp(1)) = 47.5%`。

x0 | x1 | p(x) (numerical) | p(x) (theoretical)
---|----|------------------|-------------------
0  | 0  |  17.3935         | 17.5
0  | 1  |  17.5964         | 17.5
1  | 0  |  17.6605         | 17.5
1  | 1  |  47.3495         | 47.5

### W = 0, b = [1 0] のとき

`p(x_0 = 1, x_1 = 1) = exp(1) / 2(exp(0) + exp(1)) = 36.55%`。

x0 | x1 | p(x) (numerical) | p(x) (theoretical)
---|----|------------------|-------------------
0  | 0  |  13.4201         | 13.45
0  | 1  |  13.5651         | 13.45
1  | 0  |  36.7111         | 36.55
1  | 1  |  36.3037         | 36.55
